# 金橙V 升级标准

## 橙V 升级标准

| 条件 | 要求 |
|------|------|
| 认证状态 | 已认证黄V |
| 铁粉数 | ≥ 100 |
| 近30天排水阅读量 | ≥ 30 万 |

## 金V 升级标准

| 条件 | 要求 |
|------|------|
| 认证状态 | 已认证黄V |
| 粉丝量 | ≥ 1 万 |
| 铁粉数 | ≥ 1000 |
| 近30天排水阅读量 | ≥ 1000 万 |

---

## 升级路径说明

- **黄V → 橙V**：铁粉数达到 100 且近30天排水阅读量达到 30 万
- **橙V → 金V**：粉丝量达到 1 万、铁粉数达到 1000 且近30天排水阅读量达到 1000 万

> 以上所有路径均需已完成黄V认证。
